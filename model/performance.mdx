## Benchmark Tool

ACE-Step includes `profile_inference.py`, a comprehensive profiling and benchmarking tool that measures:

- **LLM planning time** — Token generation, CoT reasoning, CFG overhead
- **DiT diffusion time** — Per-step and total denoising time
- **VAE decode time** — Latent-to-audio conversion
- **End-to-end wall time** — Total generation time

### Quick Start

```bash
# Basic profile (single run)
python profile_inference.py

# Full benchmark matrix
python profile_inference.py --mode benchmark

# Profile with LLM thinking enabled
python profile_inference.py --thinking

# Save results to JSON
python profile_inference.py --mode benchmark --benchmark-output results.json
```

## GPU Tier Optimization

ACE-Step automatically adapts to your GPU VRAM with tier-based optimization:

| VRAM | Tier | LM Model | Optimizations | Max Batch | Max Duration |
|------|------|----------|---------------|-----------|-------------|
| **≤4GB** | tier1 | None | INT8 + full CPU offload + VAE on CPU | 1 | 120s |
| **6GB** | tier2 | None | INT8 + CPU offload + tiled VAE (256) | 1 | 180s |
| **8GB** | tier4 | `0.6B` (pt) | Lightweight LM, INT8 optional | 2 | 240s |
| **12GB** | tier5 | `1.7B` (vllm) | vLLM backend, minimal offload | 2 | 600s |
| **16GB** | tier6a | `1.7B` (vllm) | Offload when needed | 4 | 600s |
| **20GB** | tier6b | `1.7B` (vllm) | No offload required | 4 | 600s |
| **24GB+** | unlimited | `4B` (vllm) | All models fit, max performance | 8 | 600s |

<Note>
  The UI automatically selects the optimal configuration for your GPU. All settings are tier-aware and pre-configured.
</Note>

## Quality vs Speed Trade-offs

### Maximum Speed Configuration

```python
params = GenerationParams(
    caption="upbeat electronic dance music",
    thinking=False,  # No LM overhead
    inference_steps=8,  # Minimum steps
)

config = GenerationConfig(
    batch_size=1,  # Single generation
)

# Use turbo + no LM
# Result: < 2s on A100, < 7s on RTX 3090
```

### Maximum Quality Configuration

```python
params = GenerationParams(
    caption="emotional piano ballad with strings",
    thinking=True,  # LLM planning
    use_cot_metas=True,  # LLM infers metadata
    use_cot_caption=True,  # LLM optimizes caption
    inference_steps=50,  # More refinement
    guidance_scale=7.0,  # CFG (SFT/Base only)
)

# Use SFT or Base + 4B LM
# Result: 15-40s depending on hardware, best quality
```

### Balanced Configuration (Recommended)

```python
params = GenerationParams(
    caption="...",
    thinking=True,
    inference_steps=8,
)

config = GenerationConfig(
    batch_size=2,
)

# Use turbo + 1.7B LM (default)
# Result: 3-10s depending on hardware, excellent quality
```
