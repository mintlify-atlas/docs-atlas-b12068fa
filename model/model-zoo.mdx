## Turbo Series

<Note>
  **Recommended for daily use** — The Turbo series delivers commercial-grade quality in just 8 diffusion steps, making generation 6x faster than 50-step models.
</Note>

### acestep-v15-turbo (Default)

<Card title="Default Recommended Model" icon="star">
  The best balance of quality, speed, and creativity for most use cases.
</Card>

**Key Features:**
- **8 inference steps** — Under 2s per song on A100, under 10s on RTX 3090
- **Joint distillation** — Trained on shift values 1, 2, 3 for optimal balance
- **No CFG required** — Distilled to follow prompts without guidance overhead
- **Thoroughly tested** — Most stable and reliable variant

**Best For:**
- Daily music generation
- Rapid prototyping
- Batch generation
- Low-latency applications

```bash
# Download turbo model
uv run acestep-download --model acestep-v15-turbo
```

### Turbo Variants

The turbo series includes specialized variants distilled with different shift configurations:

<Tabs>
  <Tab title="turbo-shift1">
    **Distilled with shift=1**
    
    - More details and texture
    - Richer orchestration
    - Weaker semantic adherence
    - Best for: Textured, complex arrangements
  </Tab>
  <Tab title="turbo-shift3">
    **Distilled with shift=3**
    
    - Clearer overall structure
    - Stronger prompt following
    - Richer timbre, but may sound "dry"
    - Minimal orchestration complexity
    - Best for: Simple, clean production
  </Tab>
  <Tab title="turbo-continuous">
    **Experimental: Continuous shift 1-5**
    
    - Supports dynamic shift tuning at inference
    - Most flexible configuration
    - Less thoroughly tested
    - Best for: Advanced users experimenting with shift
  </Tab>
</Tabs>

<Note>
  **What is shift?** Shift determines timestep allocation during diffusion. Higher shift = more focus on early denoising (building structure), lower shift = more even distribution (refining details).
</Note>

## SFT Model

### acestep-v15-sft

**50-step model with Classifier-Free Guidance support**

```python
params = GenerationParams(
    caption="dreamy synthwave ballad",
    inference_steps=50,
    guidance_scale=7.0  # CFG supported
)
```

**Key Features:**
- **CFG tuning** — Adjust `guidance_scale` (1.0-15.0) for prompt adherence
- **50 inference steps** — More time for detail refinement
- **Better semantic parsing** — Compared to turbo, better at complex prompts
- **Supervised fine-tuning** — Enhanced instruction following

**Trade-offs:**
- Slower generation (6x more steps)
- Slight quality reduction from error accumulation
- More VRAM usage during inference

**Best For:**
- Users who don't mind longer inference time
- Fine-tuning CFG for specific needs
- Complex prompt interpretation
- Production workflows prioritizing quality over speed

```bash
# Download SFT model
uv run acestep-download --model acestep-v15-sft
```

### Choosing an LM

<Tabs>
  <Tab title="No LM">
    **Disable thinking mode**
    
    **Speed:** ⚡⚡⚡⚡ Fastest
    
    **Best For:**
    - Cover mode (audio reference provides structure)
    - You provide all metadata manually
    - Maximum generation speed
    - VRAM < 4GB
    
    ```python
    params = GenerationParams(
        thinking=False,  # Disable LM
        caption="...",
        bpm=120,  # Manual metadata
        keyscale="C Major"
    )
    ```
  </Tab>
  <Tab title="0.6B">
    **acestep-5Hz-lm-0.6B**
    
    **Speed:** ⚡⚡⚡ Fast  
    **World Knowledge:** Basic  
    **Memory:** Weak
    
    **Best For:**
    - VRAM 6-8GB
    - Rapid prototyping
    - Simple prompts
    - Low-resource environments
    
    ```bash
    uv run acestep-download --model acestep-5Hz-lm-0.6B
    ```
  </Tab>
  <Tab title="1.7B (Default)">
    **acestep-5Hz-lm-1.7B**
    
    **Speed:** ⚡⚡ Medium  
    **World Knowledge:** Medium  
    **Memory:** Medium
    
    **Best For:**
    - VRAM 8-16GB (default recommendation)
    - Balanced quality and speed
    - General-purpose use
    - Most users
    
    ```bash
    uv run acestep-download --model acestep-5Hz-lm-1.7B
    ```
  </Tab>
  <Tab title="4B">
    **acestep-5Hz-lm-4B**
    
    **Speed:** ⚡ Slower  
    **World Knowledge:** Rich  
    **Memory:** Strong
    
    **Best For:**
    - VRAM > 16GB
    - Complex compositions
    - Long-tail styles/instruments
    - Best quality generation
    - Reference audio melody copying
    
    ```bash
    uv run acestep-download --model acestep-5Hz-lm-4B
    ```
  </Tab>
</Tabs>

### VRAM Requirements

| GPU VRAM | Recommended LM | Backend | Notes |
|----------|---------------|---------|-------|
| **≤6GB** | None (DiT only) | — | LM disabled; INT8 quantization + CPU offload |
| **6-8GB** | `0.6B` | `pt` | Lightweight LM with PyTorch backend |
| **8-16GB** | `0.6B` / `1.7B` | `vllm` | 0.6B for 8-12GB, 1.7B for 12-16GB |
| **16-24GB** | `1.7B` | `vllm` | 4B available on 20GB+; no offload on 20GB+ |
| **≥24GB** | `4B` | `vllm` | Best quality, all models fit without offload |

## Model Selection Guide

### By Use Case

| Use Case | DiT Model | LM Model |
|----------|-----------|----------|
| **Daily music generation** | `turbo` | `1.7B` |
| **Maximum speed** | `turbo` | No LM or `0.6B` |
| **Best quality** | `sft` or `turbo-rl` | `4B` |
| **Track separation** | `base` | `1.7B` |
| **Multi-track composition** | `base` | `1.7B` |
| **Custom fine-tuning** | `base` | — |
| **Low VRAM (< 4GB)** | `turbo` + INT8 | No LM |
| **Complex prompts** | `sft` | `4B` |

### Download Commands

```bash
# Download default models (turbo + 1.7B LM)
uv run acestep-download

# Download all models
uv run acestep-download --all

# Download specific model
uv run acestep-download --model acestep-v15-base
uv run acestep-download --model acestep-5Hz-lm-4B

# List available models
uv run acestep-download --list
```
