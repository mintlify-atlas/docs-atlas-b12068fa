## Overview

ACE-Step 1.5 is a highly efficient open-source music foundation model that brings commercial-grade generation to consumer hardware. On commonly used evaluation metrics, ACE-Step 1.5 achieves quality beyond most commercial music models while remaining extremely fast—under 2 seconds per full song on an A100 and under 10 seconds on an RTX 3090.

The model runs locally with less than 4GB of VRAM and supports lightweight personalization through LoRA training from just a few songs.

## Advanced Capabilities

ACE-Step 1.5 unifies precise stylistic control with versatile editing capabilities:

- **Reference Audio Input** — Use reference audio to guide generation style
- **Cover Generation** — Create covers from existing audio
- **Repaint & Edit** — Selective local audio editing and regeneration
- **Track Separation** — Separate audio into individual stems
- **Multi-Track Generation** — Add layers like Suno Studio's "Add Layer" feature
- **Vocal2BGM** — Auto-generate accompaniment for vocal tracks
- **Metadata Control** — Control duration, BPM, key/scale, time signature
- **Audio Understanding** — Extract BPM, key/scale, time signature & caption from audio
- **LRC Generation** — Auto-generate lyric timestamps for generated music

## Getting Started

<Steps>
  <Step title="Install uv package manager">
    Install the uv package manager for Python dependency management:
    
    ```bash
    curl -LsSf https://astral.sh/uv/install.sh | sh
    ```
  </Step>
  
  <Step title="Clone and install">
    Clone the repository and install dependencies:
    
    ```bash
    git clone https://github.com/ACE-Step/ACE-Step-1.5.git
    cd ACE-Step-1.5
    uv sync
    ```
  </Step>
  
  <Step title="Launch">
    Launch the Gradio UI or API server:
    
    <CodeGroup>
    ```bash Gradio UI
    uv run acestep
    ```
    
    ```bash API Server
    uv run acestep-api
    ```
    </CodeGroup>
    
    Models auto-download on first run.
  </Step>
</Steps>
